🛡 Defense-in-Depth Architecture

Project Aura implements security at every layer of the stack — from network perimeter through application logic, AI agent behavior, and supply chain integrity. The architecture addresses both traditional application security (OWASP Top 10) and emerging AI-specific threats (prompt injection, agent confusion, GraphRAG poisoning, model weight exfiltration).

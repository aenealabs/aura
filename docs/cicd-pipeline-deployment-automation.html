Pipeline Architecture Overview

Project Aura uses an enterprise-grade, modular CI/CD architecture powered by AWS CodeBuild and CloudFormation. The system manages 24 CodeBuild projects across a 9-layer deployment cascade with 3-gate validation, automated versioning via Release Please, and comprehensive security controls at every stage.

Single Source of Truth
CodeBuild is the ONLY authoritative deployment method. All manual deployments are prohibited to maintain audit trail consistency and IAM integrity. If a manual deployment occurs, the stack must be deleted and redeployed via CodeBuild.

Public Registries Prohibited

Build Pattern

Local Development (Podman-first)

Podman is primary runtime to avoid Docker Desktop licensing fees (ADR-049).

CI/CD Build Flow

  1. Get AWS account ID via aws sts get-caller-identity
  2. Construct private base image URI
  3. Build with --build-arg PYTHON_BASE_IMAGE=${PRIVATE_BASE}
  4. Tag and push to private ECR
  5. K8s admission controller verifies image signature

Always use --platform linux/amd64 on Apple Silicon.

Key Deployment Patterns

Idempotent CloudFormation Deploy
aws cloudformation deploy \ --stack-name ${PROJECT}-<layer>-${ENV} \ --template-file deploy/cloudformation/<template>.yaml \ --parameter-overrides <params> \ --capabilities CAPABILITY_NAMED_IAM \ --tags Project=$PROJECT Environment=$ENV \ --no-fail-on-empty-changeset \ --region $AWS_DEFAULT_REGION
  • --no-fail-on-empty-changeset: Safe for CI/CD (no error when no changes)
  • CAPABILITY_NAMED_IAM: Required for custom IAM resource names
  • Tags for cost allocation and governance
Stack State Machine Handling

Critical CI/CD Rules

Prohibitions
No Manual DeploymentsBreaks audit trail and IAM consistency. Fix: delete stack → redeploy via CodeBuild.
No Duplicate BuildsNever trigger CodeBuild while another build is running. Check: list-builds-for-project
No Hardcoded SecretsUse Secrets Manager or SSM. Pre-commit hook blocks credential commits.
No Hardcoded ValuesAlways use ${AWS::AccountId}, !Sub, !Ref. Never hardcode account IDs or endpoints.
No Public ImagesAll container builds MUST use private ECR base images.
Requirements
70% Test CoverageEnforced in pyproject.toml. Never lower. Add tests instead.
600-Line Buildspec MaxSplit into sub-buildspecs when approaching limit.
10 KB IAM Policy MaxUse managed policies for large permission sets.
Conventional CommitsRequired for Release Please: feat:, fix:, docs:, etc.
GovCloud PartitionsAlways use ${AWS::Partition}. Never hardcode arn:aws.
